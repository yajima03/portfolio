# Portfolio Site
就職活動用に作成した、個人ポートフォリオサイトです。  
研究内容、開発経験、論文・発表実績、技術スタック、学生時代の取り組みなどを、1つのサイト上で分かりやすく整理して掲載しています。

## 公開URL
- Portfolio: https://portfolio-yajima03s-projects.vercel.app/

## 概要
このサイトは、自分の研究・開発経験・実績を短時間で把握してもらえるように構成したポートフォリオです。

本サイトでは、それらの内容を以下のようなセクションに整理して掲載しています。

- プロフィール
- 研究概要
- 論文・発表
- 制作物・開発プロジェクト
- 経験・取り組み
- 技術スタック
- 資格・アルバイト経験
- 将来のエンジニア像
- 外部リンク・連絡先


## 使用技術
- React
- TypeScript
- Vite
- Tailwind CSS
- Vercel
- GitHub


## ローカルでの実行方法

### 1. 依存関係のインストール
```bash
npm install
```
### 2. 開発サーバの起動
```bash
npm run dev
```
### 3.本番ビルド
```bash
npm run build
```

## 工夫した点
- 研究・開発経験・実績が短時間で把握できるよう、情報をセクションごとに整理した
- Webサイトとして公開し、採用担当者がURLからすぐ閲覧できるようにした
- GitHub と Vercel を連携し、修正内容を反映しやすい構成にした
- 技術的な内容だけでなく、強み・弱み、興味関心、学生時代の経験も掲載し、人柄が伝わる構成を意識した
